# PROJECT_DEMOQA_CI

Este proyecto es un ejemplo de automatización de pruebas para la página DemoQA utilizando Selenium WebDriver y pytest. 

## Requisitos

Antes de comenzar, asegúrate de tener los siguientes requisitos instalados:

- Python 3.7 o superior
- pip (gestor de paquetes de Python)
- Un entorno virtual (opcional pero recomendado)

## Instalación

Sigue estos pasos para clonar el repositorio y configurar el entorno de desarrollo:

```bash
# Clonar el repositorio
git clone https://github.com/usuario/PROJECT_DEMOQA_CI.git

# Navegar al directorio del proyecto
cd PROJECT_DEMOQA_CI

# Crear y activar un entorno virtual (opcional)
python -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate

# Instalar las dependencias
pip install -r requirements.txt
